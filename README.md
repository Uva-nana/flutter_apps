# 📝 Word Quiz App                                                                                                                                                                                                                                                                                                  
  A fun and interactive multiple-choice quiz app built with Flutter. Test your knowledge in Flutter, Python, and General topics — with a timer, score        tracking, and detailed result review.  
  Try the app here: [https://quizapp-b3f13.web.app](https://quizapp-b3f13.web.app)
  ---                                                                                                                                                      
                                                                                                                                                           
  ## ✨ Features

  - 🗂️ **3 Categories** — Flutter, Python, General Knowledge
  - 🎯 **3 Difficulty Levels** — Easy, Medium, Hard
  - ⏱️ **15-Second Timer** — auto-moves to next question if time runs out
  - 🏆 **High Score Tracking** — saves best score per category and difficulty
  - ❌ **Wrong Answer Review** — see correct answers for every mistake
  - 📤 **Share Your Score** — copy result to clipboard and share
  - 🎨 **Beautiful UI** — warm gradient theme with glass-morphism card design
  - 🔄 **Smooth Animations** — fade transitions between questions

  ---

  ## 📱 Screens

  | Screen | Description |
  |---|---|
  | Home | Choose a quiz category and see your best score |
  | Difficulty | Pick Easy / Medium / Hard |
  | Quiz | Answer questions with live timer and score |
  | Results | View score, wrong answers, and share result |

  ## 🚀 Getting Started

  ### Prerequisites
  - Flutter SDK `^3.11.4`
  - Dart SDK (comes with Flutter)

  ### Run the App

  ```bash
  git clone https://github.com/Uva-nana/word_quiz_app.git
  cd word_quiz_app
  flutter pub get
  flutter run

  Supports Android, iOS, Web, Windows, macOS, Linux

  ---
  🎮 How to Play

  1. Select a category — Flutter, Python, or General
  2. Choose a difficulty — Easy (5 Qs), Medium (8 Qs), Hard (10 Qs)
  3. Answer before the 15-second timer runs out
  4. See your final score and review wrong answers
  5. Beat your high score and try again!
